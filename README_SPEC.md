# 反手一个宠物 MVP 规格

## 形态
- 纯 HTML5 Canvas 单页
- 入口文件：`index.html`
- 离线运行，无网络请求、无外链、无 iframe
- 竖屏优先，适配移动端 9:16

## 核心循环
1. 进入领养台
2. 获得蛋壳券
3. 翻转手机或长按蛋壳
4. 孵出宠物
5. 播放语音字幕
6. 收养进图鉴
7. 分享声波卡
8. 等待蛋壳券恢复

## MVP 功能
- 主界面领养台
- 翻转/长按孵化
- 随机宠物生成
- 稀有度展示
- 宠物语音字幕
- 图鉴列表与详情
- 分享卡页面
- 本地存档

## 资源与包体
- 资源全部本地打包
- 宠物图只保留 8 只代表素材
- 稀有度图保留 4 张
- 主界面图、分享卡图各 1 张
- 目标总包体控制在 8MB 以内

## 已落地资源目录
- `assets/maininterface.png`
- `assets/sharingcard.png`
- `assets/pets/*.png`
- `assets/rarity/*.png`

## 说明
- 桌面端或无传感器时可用长按蛋壳兜底
- 数据保存在 `localStorage`
- 语音使用浏览器 `speechSynthesis` 作为轻量方案
